# TechBazar Inventory Project

## Concepts Learned
* Creating tables with strict constraints (`SERIAL`, `PRIMARY KEY`, `CHECK`).
* Bulk inserting records.
* Data manipulation (`UPDATE`).
* Data Retrieval (`WHERE`, `ORDER BY`, `LIMIT`).
* Basic Aggregation and Grouping (`SUM()`, `COUNT()`, `GROUP BY`).

## Key Takeaway
Learned that calculating total inventory value requires multiplying columns inside the aggregation function: `SUM(price * stock)`.
